1.    Introduction

TLI CMS is a content management system (CMS) which you can use to build a web site. It is made easy to contribute content to a website with the help of this CMS.

2.    How to go about it

Click on the Admin Link in your website & Log in with your valid credentials. If you are having ‘Super Admin’ rights you will be having full access rights to add/ edit/ delete everything on this.

3.    User Management

      If you have Super Admin rights you will have full access rights to add/ edit/ delete user. For users with normal admin rights, the User Management module will be absent.

 

1.       Click on User Management module or link from the tool bar. It will lead you to the user list with add/ edit, delete options available. It also shows the Email, Level and status of the user.

 

You can also search the user list by name or email or level.

2.       Click on Edit link will lead to edit form with the fields filled in for edit.

3.       Click on Delete link will delete the user once after confirmation.

4.       Click on Add New User link will lead to add form to be filled in and submitted for user creation.

5.      If you have Super Admin rights you will have full access rights to add/ edit/ delete User.

6.      For users with normal admin rights, the User Management module will be absent.

 

ADD NEW USER

     This modules helps super administrator to add/edit/delete site administrator. This section is only available for super

      administrator.

 

      FIELDS

                 1) Name           : Super administrator has to enter name of user.

                 2) Accesslevel : Super administrator has to set the access level of newly created user.

                                                  Accesslevel decides the privileges allowed to this user.

                                                  Super admin has maximum privileges

                                                  Administrator has lower privileges than super admin.

                  3) Email          : Super admin has to provide email address of new user. 

                  4) Username : This field will be used for login username.                           

                                                   This field is unique.

                  5) Password   :   Super admin has to provide Password for new user.

 

 

4.    Content Management

       Admin users can update site content.

1.      Click on Content Management will take you to Page List with add/ edit, delete options available. It shows page name and page title.

 

You can also search the list by Page Name or Page Title.

2.       Click on Edit link will lead to edit form with the fields filled in for edit.

3.       Click on Delete link will delete the page once after confirmation.

4.       Click on Add New Page link will lead to add form to be filled in and submitted for page creation.

 

Add new content

           This module helps site administrator and super administrator to add dynamic content.


          FIELDS

          1) Name          : user has to provide a name while creating a page. This name should be unique.

          2) Title             : This text will display as title of the browser.

          3) Content      : Content will be displayed on body part.

          4)Keyword      : This data will display on meta tag keyword section.

          5) Description: This data will display on meta tag description section.

 

5.    Menu Management

1.        Click on Menu Management will take you to Menu List with add/ edit, delete options available. It shows page name and page title.

You can also search the list by menu name.

2.       Click on Edit link will lead to edit form with the fields filled in for edit.

3.       Click on Delete link will delete the page once after confirmation.

4.       Click on Create New Menu will lead to Add form to be filled in and submitted.

  ADD NEW MENU

This module helps administrator to create top menu for frontend section.


          FIELDS

1) Link Type      : This field decides whether newly created menu will link to an internal page or an external website.

2) Parent Menu: This field describes whether the menu should be a parent menu or a submenu. If the menu is a parent menu select -- Main Menu -- in the drop-down. If the menu is sub-menu select the parent menu from the drop-down.

3) Menu Label: This text will display on menu.

4) Menu Link    : This link takes either internal page link or an external URL.

5) Publish        : This fields makes a menu visible or hidden as per requirement.                               

Depending on the choice you make here, the menu will be External/ Internal or Published or Hidden.

  ADD NEWS

7. Categories List

1. Click on Categories, it will show you the all categories with delete/edit options available. It shows category name and description.

           
     Search by Name and Description is also possible.

          2.Click on Delete will delete the category after confirmation.

      

3. Click on Add New Category will lead to add form to be filled in and submitted.

    ADD NEW CATEGORY

    Admin can upload Category images with help of this module.

          FIELDS

     1)Type : Type of the category.

     2)Name: Name of the category

     3)Description: Description of category.

     4)Image: Image of ategory.

 

     8. Products List

Click on Products will show all products with add/ edit, delete options available.  Click on image icon to view the image of that product.

            
     Search by Name and Description is also possible.

1. Click on Edit link will lead to edit form with the fields filled in for edit.

2. Click on Delete link will delete the Product once after confirmation.

3. Click on Add New Product will lead to add form to be filled in and submitted.

    ADD NEW PRODUCTS

    Admin can upload Product images with help of this module.

          FIELDS

     2)Category: Category of the product

     3)Price: Price of product.

     4)Description: Description of product.

     5)Long Description: Long description of product.

     7)Image: Image of product.
